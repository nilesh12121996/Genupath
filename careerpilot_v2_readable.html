Human behavior engine new — cross-cutting service used by Agents 1, 4, 5

Every browser action indistinguishable from a real person

Fingerprint & identity

playwright-extra stealth · residential proxy (Brightdata sticky session) · canvas randomization

  • playwright-extra + stealth plugin hides webdriver flag, normalizes headless JS APIs
  • Residential proxy rotation — sticky session per domain (same IP for full session)
  • Randomized viewport: 1366×768, 1440×900, or 1920×1080
  • Locale, timezone, accept-language match proxy geo
  • Canvas fingerprint randomization

Behavioral timing

Gaussian delays (μ=1.2s σ=0.4s) · reading time simulation · business-hours gate (8am–9pm)

  • All waits drawn from Gaussian distribution — never fixed delays
  • Page reading time = word_count / 200 WPM before any action
  • Business hours gate: actions only 8am–9pm your timezone
  • Random micro-pauses 50–300ms mid-form (simulates thinking)

Mouse & keyboard sim

Bezier mouse paths · 45–65 WPM typing with occasional typo + backspace corrections

  • Mouse moves along Bezier curves — never teleports to coordinates
  • Typing speed 45–65 WPM drawn from distribution, not instant .fill()
  • 1-in-50 chance of typo + backspace correction per field
  • Tab between fields, random scroll while "reading" before applying

Session warm-up & CAPTCHA gap fixed

Pre-application browsing session · 2captcha fallback · human-in-the-loop Slack alert

  • Before applying: visit homepage, scroll, click 1–2 nav links to build real session history
  • CAPTCHA hit → 2captcha/Anti-Captcha API (1st retry)
  • CAPTCHA fails twice → pause agent + fire Slack alert + await manual clearance
  • Session cookies persisted and reused per domain (stored in Vault)
