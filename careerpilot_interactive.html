Cross-Cutting Service

Human behavior engine — 6 layers of deception

Click Next to activate each layer. All 6 must work together — any one missing gets you flagged.

playwright-stealth + TLS fingerprint
Masks navigator.webdriver, chrome.runtime, WebGL renderer, AudioContext. TLS/JA3 randomization defeats Cloudflare at the network layer — something stealth alone cannot do.
Residential proxy (sticky session)
Webshare residential proxy — same home IP for the entire session per domain. Data-center IPs (like Browserbase's default) get flagged by LinkedIn immediately. Sticky = same IP, not rotating mid-session.
192.168.x.x → flagged
Gaussian timing engine
All delays drawn from Gaussian distribution (μ=1.4s, σ=0.5s) — never fixed intervals. Business hours gate: 8am–7pm only, no weekends. Rage-click prevention: >800ms between same-element clicks.
Bezier mouse paths + typo simulation
Mouse curves to target with ~20% overshoot + correction. Typing at 45–65 WPM with per-character variance. 1-in-40 chance of typo + backspace correction per field.
Aged browser profiles (6-month warm-up)
Fresh profiles are the #1 bot signal. Each domain gets a persistent Playwright context stored on disk — 2-week warm-up of browsing before first real application. History, cookies, local storage all intact.
profile age: 0 days
CAPTCHA handler + human fallback
2captcha API for hCaptcha/reCAPTCHA (~15s solve time). reCAPTCHA v3 (score-based) handled by session aging — behavioral warm-up builds the score. On second failure: pause agent + Slack alert + await human.
2captcha solve time: ~15s
No single layer is enough. Cloudflare Bot Management and Datadome check the TLS fingerprint, IP reputation, mouse entropy, and behavioral timing simultaneously. The moment any layer is missing, the score drops and you get challenged or blocked.
