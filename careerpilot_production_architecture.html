CareerPilot — Production Architecture

Practical AI‑Driven Job Application Automation System

Tier 1 — Control Plane

Workflow Orchestrator Core
Central engine coordinating agents and managing application lifecycle.
  • LangGraph or Temporal workflow engine
  • Tracks Job ID states
  • Handles retries and pipeline branching
  • Controls daily application limits
Profile Vault
Secure storage for master resume, work history and personal information.
  • Structured JSON experience database
  • Reusable EEO responses
  • Credential encryption
  • Resume base templates
Observability
Monitoring system performance, cost, and application success rate.
  • Token spend tracking
  • Application conversion analytics
  • Error logging and tracing
  • Daily cost guardrails

Tier 2 — Job Discovery Pipeline

Job Scraper Network Pipeline
Scheduled collectors gather job listings from multiple job boards.
  • LinkedIn scraping
  • Greenhouse / Lever APIs
  • Indeed crawler
  • Runs every 10–20 minutes
Job Normalizer
Transforms raw job descriptions into structured schema.
  • Extract skills and tech stack
  • Detect role seniority
  • Location parsing
  • Salary extraction
Duplicate Detector
Prevents applying to the same job across different platforms.
  • Company + title hash
  • URL similarity detection
  • Job board cross‑reference

Tier 3 — Intelligent Filtering

Visa Feasibility Scorer
Estimates sponsorship probability using historical datasets.
  • LCA dataset analysis
  • Employer sponsorship history
  • Probability scoring model
Job Ranking Engine
Scores opportunities based on match quality.
  • Skill similarity
  • Salary weighting
  • Commute distance
  • Visa likelihood

Tier 4 — Resume Intelligence

Resume Tailoring Engine
Generates job‑specific resume versions from master experience data.
  • Semantic bullet retrieval
  • Keyword alignment
  • Template‑based PDF rendering
  • Maximum 2 rewrite iterations
Human Review Gate
Approval checkpoint before submitting applications.
  • Dashboard review interface
  • Edit resume before submission
  • Approve / Reject pipeline step

Tier 5 — Application Automation

Application Bot
Automates form submission on supported ATS platforms.
  • Playwright automation
  • Site‑specific adapters
  • Human‑like typing delays
  • Error recovery
Networking Assistant
Suggests outreach messages to recruiters and hiring managers.
  • LinkedIn profile discovery
  • AI‑generated intro messages
  • Human sends connection request

Tier 6 — Data Infrastructure

Application Database Infra
Persistent storage for job pipeline and application results.
  • PostgreSQL state database
  • Application history
  • Interview tracking
Vector Knowledge Store
Semantic memory for resume bullets and interview answers.
  • Embedding search
  • Experience retrieval
  • Question‑answer archive
Document Storage
Archive for generated resumes and application artifacts.
  • S3 object storage
  • Versioned resume PDFs
  • Application attachments
